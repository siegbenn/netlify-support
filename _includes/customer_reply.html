Hi insert_name,

Sorry your site isn’t building! To help us get this fixed, it would be great if you could look into a few things for us:

  1. What build tool and version are you using?

    2. A lot of times, a build will fail due to the version of the build tool specified in your project. The version you are using should be located in your package file (package.json, requirements.txt or Gemfile).

  2. Double check that your build command doesn’t serve content.

    3. If your build command is trying to serve your site ($ node index.js or $ npm start), your build will probably timeout. We will always try to run your build command but if it doesn’t complete in 15 minutes, we shut it down.

  3. Are all of the packages your project uses public?

    4. Our build containers have access to all public repositories and packages. But if you are using something like a private repository or npm package, we won’t be able to access it. If that’s the case we recommend a local copy of the private repository or package before sending us a build.

If everything above looks good and your site still isn’t building, don’t worry! We’re always happy to dig into the logs with you.
